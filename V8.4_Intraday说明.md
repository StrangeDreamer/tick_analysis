# V8.4-Intraday 日内优化版本说明

## 🎯 版本定位

**V8.4-Intraday** 是专为日内短线交易设计的优化版本，100%聚焦tick数据中的日内特征。

---

## 🚀 核心改进

### 1. 相对净买入（最重要）

**改进前（V8.3）：**
```python
# 使用绝对净买入量
net_buy_score = 净买入量 / 100万 × 35
```

**问题：**
- 大盘股（茅台）日成交5000万手，净买入100万 → 只占2%
- 小盘股日成交50万手，净买入10万 → 占20%（更强！）
- 但评分却是：茅台 +3.5分 vs 小盘股 +0.35分 ❌

**改进后（V8.4）：**
```python
# 使用相对净买入
relative_net_buy = 净买入量 / 总成交量
net_buy_score = relative_net_buy × 175
```

**效果：**
- 茅台：净买入100万 / 5000万 = 2% → +3.5分
- 小盘股：净买入10万 / 50万 = 20% → +35分（满分！）✅
- **大小盘股公平对比！**

---

### 2. 流动性评估（日内必备）

**新增功能：**
```python
def _evaluate_liquidity(total_volume, tick_count):
    日成交量 < 10万手  → -20分（严重不足）
    日成交量 < 30万手  → -10分（中度不足）
    tick笔数 < 500笔   → -5分（轻度不足）
    日成交量 > 100万手 → +5分（流动性优秀）
```

**作用：**
- 自动过滤"没人交易"的冷门股
- 确保买得进、卖得出
- 日内交易的生命线！

---

### 3. 动量加速度（捕捉爆发力）

**算法：**
```python
# 将tick数据分成5个时段
时段1: 9:30-10:00
时段2: 10:00-10:45
时段3: 10:45-11:30/13:00-13:30
时段4: 13:30-14:15
时段5: 14:15-15:00

# 计算加速度
acceleration = 时段5涨幅 - 时段1涨幅
```

**评分：**
- 加速度 > 0 → 越涨越快 → +10分 ✅
- 加速度 < 0 → 冲高回落 → -10分 ❌

**效果：** 识别"开盘冲高后继续拉升"的强势股！

---

### 4. 上涨持续性（避免假突破）

**算法：**
```python
# 统计连续上涨和连续下跌的tick数
平均上涨持续 = 3笔tick
平均下跌持续 = 1笔tick

# 持续性比率
sustainability = 3 / 1 = 3.0

# 评分
sustainability_score = (3.0 - 1.0) × 10 = +20分（但限制在±10分）
```

**效果：**
- 持续性 > 1 → 上涨比下跌持久 → 加分 ✅
- 持续性 < 1 → 下跌比上涨持久 → 扣分 ❌

**过滤："快速拉高然后阴跌"的假突破！**

---

### 5. 尾盘权重提升

**改进：**
- 收盘动量评分：15分 → **20分** ⚡
- 超额收益评分：15分 → **10分**（降低）

**原因：**
- 日内交易更关注尾盘走势
- 尾盘强势 → 第二天高开概率大
- 超额收益是结果，不是原因

## 🎯 适用场景

### ✅ 最适合

1. **日内短线交易**
   - T+0思维（A股做T）
   - 当日买入，尾盘或次日卖出
   - 持仓时间：几小时到1天

2. **强势市场**
   - 大盘上涨 > 1%
   - 涨停板数量多
   - 市场情绪高涨

3. **盘中监控**
   - 09:35 第一次运行（开盘）
   - 10:30 第二次运行（确认）
   - 14:00 第三次运行（午盘）
   - 14:45 第四次运行（尾盘）

---

### ⚠️ 不适合

1. **中长线投资**
   - 需要基本面分析
   - 需要趋势判断
   - 建议看日K线和财报

2. **震荡市/熊市**
   - 个股难独善其身
   - 假突破增多
   - 建议降低仓位或观望

3. **新手投资者**
   - 日内交易风险高
   - 需要快速决策
   - 建议先纸上交易1-2个月

## 💡 使用建议

### 评分解读

```
90-100分：⭐⭐⭐⭐⭐ 极度强势，考虑重仓（30-50%）
70-89分：⭐⭐⭐⭐   强势，考虑中仓（20-30%）
50-69分：⭐⭐⭐     一般，考虑轻仓（10-20%）
30-49分：⭐⭐       观察，考虑试探（5-10%）
0-29分：⭐         弱势，不建议买入
负分：❌           明显弱势，考虑卖出
```

### 风险控制

1. **止损**
   - 买入后跌破 -3% → 立即止损
   - 不要幻想"回本"

2. **止盈**
   - 买入后涨幅 > 5% → 移动止盈到 2%
   - 买入后涨幅 > 8% → 移动止盈到 5%

3. **仓位管理**
   - 单只股票不超过 30%
   - 建议持仓 3-5 只
   - 总仓位不超过 80%

4. **时间管理**
   - 开盘30分钟内不追高
   - 收盘前15分钟不抄底
   - 午休时间不操作
